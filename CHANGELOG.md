# Внесені виправлення

1. З AI-зображення та engagement-зображення прибрано сторонні фрагменти тексту.
2. Мозаїку можливостей замінено готовим зображенням `feature-cloud.png` з правильними іконками та розкладкою.
3. Для кнопок реалізовано об'ємний ефект із нижньою тінню та станом натискання.
4. Текстовий логотип у header і footer замінено SVG-зображеннями.
5. Назви компаній у секції довіри замінено SVG-логотипами.
6. У футері встановлено правильні іконки LinkedIn, YouTube, Twitter та Dribbble.
7. `overflow-x: hidden` не використовується на `html` або `body`; ширина контролюється локально через адаптивні обмеження.
8. Додано `.nojekyll` і workflow GitHub Pages для стабільного деплою статичного сайту.
