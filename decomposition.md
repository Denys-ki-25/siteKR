# Декомпозиція сайту Contrast

## 1. Загальна структура

```text
Contrast page
|-- header
|-- main
|   |-- hero
|   |-- social-proof
|   |-- ai-section
|   |-- engagement
|   |-- feature-cloud
|   `-- final-cta
`-- footer
```

Сторінка є односторінковим landing page для платформи вебінарів. Вона послідовно веде користувача від головної пропозиції до демонстрації інтерфейсу, соціального доказу, AI-функцій, інструментів залучення, мозаїки можливостей і фінального заклику до дії.

## 2. Основні компоненти

| Секція | Підкомпоненти | Призначення |
| --- | --- | --- |
| `header` | Логотип, навігація, CTA, burger-меню | Первинна навігація сайтом. |
| `hero` | Заголовок, опис, кнопки, webinar preview | Перше враження і головна цінність продукту. |
| `social-proof` | Аватари, заголовок, логотипи клієнтів | Підтвердження довіри до платформи. |
| `ai-section` | Стрічка `new`, заголовок, опис, AI preview | Пояснення AI-перепакування вебінару в контент. |
| `engagement` | Feature tabs, engagement preview | Демонстрація інструментів залучення аудиторії. |
| `feature-cloud` | Заголовок, `feature-cloud.png` | Цілісне зображення набору можливостей платформи. |
| `final-cta` | Заголовок, опис, кнопки | Повторення цільової дії наприкінці сторінки. |
| `footer` | Логотип, опис, соцмережі, службові посилання | Завершення сторінки та додаткова навігація. |

## 3. Відповідність файлам

| Файл | Роль |
| --- | --- |
| `index.html` | Семантична HTML-структура сторінки. |
| `assets/css/style.css` | Desktop-стилі, адаптивність, кнопки та локальне керування переповненням. |
| `assets/js/main.js` | Burger-меню і перемикання активного пункту engagement. |
| `assets/img/webinar-stage.png` | Зображення hero preview. |
| `assets/img/ai-preview.png` | Обрізане AI-зображення без зайвого тексту. |
| `assets/img/engagement-preview.png` | Обрізане engagement-зображення без сторонніх фрагментів. |
| `assets/img/feature-cloud.png` | Цілісна мозаїка функцій з правильними іконками. |
| `assets/img/logos/` | SVG-логотипи Contrast і клієнтів. |
| `assets/img/social/` | SVG-іконки LinkedIn, YouTube, Twitter та Dribbble. |
| `.github/workflows/pages.yml` | Деплой статичного сайту на GitHub Pages. |
