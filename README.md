# Contrast Landing Page

Правильний курсовий проєкт за варіантом Яцківа Дениса Руслановича. Джерело варіанту перевірено за файлом `Пояснювальна_записка_Яцків.docx`: макет належить сервісу Contrast.

## Структура

```text
Yatskiv_Contrast_project/
  index.html
  README.md
  decomposition.md
  assets/
    css/
      style.css
    js/
      main.js
    img/
      contrast-full.png
      webinar-stage.png
      ai-panel.png
      engagement-panel.png
      feature-cloud.png
      cta-card-ref.png
```

## Секції сторінки

- `header` - логотип Contrast, навігація, кнопки Book a demo і Start for free.
- `hero` - головний екран “Effortless webinars, powerfully engaging”.
- `social-proof` - аватари та логотипи клієнтів.
- `ai-section` - блок AI-перепакування контенту після вебінару.
- `engagement` - список сценаріїв утримання аудиторії та preview.
- `feature-cloud` - мозаїка функцій платформи.
- `final-cta` - фінальний заклик до дії.
- `site-footer` - футер із групами посилань.

## Запуск

Сайт статичний. Для перегляду відкрийте `index.html` у браузері.
